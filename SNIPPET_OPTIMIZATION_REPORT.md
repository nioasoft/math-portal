## מה זה אחוז בחשבון?

**Simplified Teaching Path**:
```
## איך מלמדים אחוזים - צעד אחר צעד:

3. הראו את הקשר לשברים: 50% = 50/100 = 1/2
4. תרגלו חישוב 10% תחילה (זה הכי קל!)
5. כשהכללים בטוחים, עברו לחישובים מורכבים יותר

**Real-World Context** (for PAA):
```
## איפה משתמשים באחוזים בחיים?

- קניות: "30% הנחה" = כמה כסף נחסוך?
- בדיקות: "את קיבלת 85%" = כמה טוב?
- כסף: ריבית בבנק נתונה כאחוז
- סטטיסטיקות: "60% מסכימים" = יחסים בחברה

## SECTION F: Content Formatting Best Practices

### Opening Paragraph Pattern (Optimized for Snippets)

**Formula**: Question + 40-60 word direct answer

```markdown
## איך מלמדים [נושא]?

[DIRECT ANSWER - 40-60 words explaining the core concept or approach]

[SUPPORTING CONTENT - numbered steps, examples, or lists]
```

**Example**:
```markdown
## איך מלמדים חיבור?

התחילו עם חפצים מוחשיים (קוביות, עפרונות) וסבירו שחיבור זה
"להוסיף" או "לאסוף יחד". השתמשו במספרים קטנים תחילה, ורק כשהילד
מוכן הציגו את הנשיאה בהדרגה.

## 5 שלבים למוצלח:
1. ...
2. ...
```

### Definition Pattern (For "What is X" Queries)

```markdown
## מה זה [concept]?

[DEFINITION - One sentence + 1-2 clarifying sentences = 30-50 words]

[CONTEXT or EXAMPLES]
```

**Example**:
```markdown
## מה זה אחוז בחשבון?

אחוז פירושו "מתוך מאה". אם אנחנו אומרים 50%, זה כמו לומר "חצי"
(50 מתוך 100). אחוז הוא בעצם שבר עם מכנה תמיד 100.

### איפה משתמשים באחוזים?
- קניות: 30% הנחה
- בדיקות: ציון של 85%
```

### Comparison Table Pattern (For Product/Concept Comparisons)

```markdown
## [Concept A] vs [Concept B] - מה ההבדל?

| מאפיין | [A] | [B] |
|--------|-----|-----|
| הגדרה | ... | ... |
| נוסחה | ... | ... |
| דוגמה | ... | ... |
| שימוש | ... | ... |
```

### FAQ Section Pattern (For PAA Optimization)

```markdown
## שאלות תכופות:

**ש: [Common Question]?**
ת: [Direct answer - 1-2 sentences]

**ש: [Related Question]?**
ת: [Direct answer - 1-2 sentences]
```
