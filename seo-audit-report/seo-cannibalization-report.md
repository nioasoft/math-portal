# SEO Keyword Cannibalization Audit Report
## Math Portal (דפי עבודה חכמים)

**Report Date:** January 17, 2026
**Audited Domain:** www.smart-worksheets.co.il
**Language:** Hebrew
**Scope:** Full site content audit for keyword overlap

---

## EXECUTIVE SUMMARY

The Math Portal demonstrates significant keyword cannibalization risks across three main areas:

1. **Grade Pages vs Topic Pages** - High overlap between grade-level content and topic-specific content
2. **Blog Posts vs Help Topics** - Duplicate coverage of same subjects with different formats
3. **Topic Pages Metadata Inconsistency** - Incomplete SEO metadata leading to unclear targeting

**Overall Risk Level:** MODERATE-HIGH

**Priority Issues:** 6 critical, 8 medium, 5 low

## SECTION 2: IDENTIFIED CANNIBALIZATION CONFLICTS

### CRITICAL CONFLICTS (High Search Volume, Direct Competition)

#### CONFLICT #1: FRACTIONS - Triple Cannibalization
**Issue Level:** CRITICAL | **Severity:** HIGH

**Competing Pages:**
1. `/fractions` (Worksheet Generator) - Primary entry point
2. `/help/fractions` (Educational Resource) - Teaching guide
3. `/blog/fractions-made-easy` (Blog Post) - Learning tips
4. `/grade/4` + `/grade/5` (Grade Pages) - Topic within curriculum

**Keyword Analysis:**
- Primary: "שברים" (Fractions)
- Long-tail: "תרגילי שברים" (Fraction exercises)
- Long-tail: "חיבור שברים" (Adding fractions)
- Long-tail: "צמצום הרחבה" (Simplification/expansion)

**Current Ranking Intent:**
- `/fractions` - Direct problem generation
- `/help/fractions` - Educational explanation
- `/blog/fractions-made-easy` - Parent-focused learning tips
- `/grade/4` & `/5` - Grade-level context

**Cannibalization Risk:** Google may consolidate ranking authority across these pages, spreading rankings instead of concentrating on one authority page. The help topic and blog post both target teaching methods, creating duplicate content angles.

**Resolution Strategy:**
- Establish `/fractions` as authority page (worksheet generator - commercial intent)
- Make `/help/fractions` a support resource (link to /fractions from help page)
- Differentiate `/blog/fractions-made-easy` as parent-focused tips (not duplicate content)
- Keep grade pages as navigation hubs (link out to /fractions)
- Implement canonical: `/help/fractions` → `/fractions` (optional, if content is similar)
- Add internal linking: Blog post → Fractions generator

## SECTION 3: GRADE PAGE OVERLAP ANALYSIS

### Issue: Grade Pages as "Hub" Pages vs Topic Pages as "Authority" Pages

**The Problem:**
Grade pages (`/grade/1` through `/grade/6`) function as navigation hubs with internal links to specific topic generators. However, they also rank for compound keywords like:

- "כיתה א חשבון" (Grade 1 math)
- "תרגול כיתה ב" (Grade 2 practice)
- "שברים כיתה ד" (Grade 4 fractions)

This creates a secondary cannibalization layer where users might land on the grade page instead of the specific topic page.

**Current Flow:**
```
/grade/4
├── Links to /fractions
├── Links to /geometry
├── Links to /decimals (via query params in curriculum)
└── Links to /word-problems
```

**SEO Impact:**
- Grade pages steal ranking authority for "כיתה ד שברים" from `/fractions` page
- Long-tail keywords split between grade + topic pages
- Each page receives fractional link equity instead of consolidated

**Resolution Strategy:**
- Grade pages should rank for: "כיתה ד חשבון", "מתמטיקה כיתה ד"
- Topic pages should rank for: "שברים", "תרגילי שברים", "חיבור שברים"
- Avoid compound "כיתה ד שברים" targeting on both pages
- Use canonical tags if content is duplicated across grade + topic
- Grade page link strategy: Use parameter-based links (`?grade=4`) to avoid authority splitting

## APPENDIX B: CONTENT DIFFERENTIATION CHECKLIST

When creating new content, use this checklist to avoid cannibalization:

**Before publishing, verify:**

- [ ] Does this content compete with existing page for primary keywords?
- [ ] What is the UNIQUE angle? (Psychology? Methods? Real-world? Parent-focused?)
- [ ] Who is the audience? (Educators, parents, students, curriculum planners?)
- [ ] Does it duplicate existing content or ADD new value?
- [ ] Which page should receive the primary keyword targeting?
- [ ] How will I link this page to existing relevant pages?
- [ ] Does the metadata reflect this unique angle?

**Approved angles for new blog posts:**
- Psychological/motivational angle (distinct from educational)
- Practical/real-world application (distinct from abstract teaching)
- Parental guidance (distinct from technical teaching)
- Curriculum context (distinct from technique teaching)
- Learning strategies/hacks (distinct from definitions)
