# תוכנית שיפורי SEO ותוכן

תאריך: 2026-06-03

## מטרות

- להפוך את דפי המשחקים לדפים סריקים עם תוכן HTML, ולא רק חוויית Canvas.
- לחזק קישורים פנימיים בין משחקים, דפי עבודה ונושאי לימוד.
- להוסיף structured data מתאים למשחקים.
- להוסיף בדיקות אוטומטיות שמונעות רגרסיות SEO בסיסיות.

## בוצע במקצה הזה

1. דפי משחקי 3D
   - נוסף תוכן HTML סריק מתחת לכל משחק.
   - נוסף הסבר מה מתרגלים במשחק.
   - נוסף תוכן לפי נושא לימוד בכל שפות האתר: עברית, אנגלית, ערבית, גרמנית, ספרדית ורוסית.
   - נוסף טקסט SEO ייחודי לכל אחד מ-44 משחקי ה-3D בכל 6 שפות האתר.
   - נוסף אזור שאלות נפוצות.
   - נוספו קישורים לדפי תרגול קשורים ולמשחקים נוספים באותו topic.

2. Structured data
   - נוסף `LearningResource` לכל דף משחק.
   - נוסף `FAQPage` לכל דף משחק.
   - נוסף `BreadcrumbList` לכל דף משחק.

3. קישורים פנימיים
   - דפי המשחקים מקשרים לדפי עבודה רלוונטיים לפי topic.
   - דפי נושא מרכזיים מקשרים חזרה למשחקים רלוונטיים:
     - חשבון
     - שברים
     - גיאומטריה
     - אחוזים
     - עשרוניים
     - יחס
     - יחידות מידה
     - סדרות
     - בעיות מילוליות

4. בדיקות אוטומטיות
   - בדיקת sitemap מוודאת שכל 44 משחקי ה-3D קיימים ב-sitemap בעברית ובאנגלית.
   - בדיקות unit ל-JSON-LD של משחקים.
   - בדיקות unit מוודאות שלכל משחק יש תוכן ייחודי בכל שפה נתמכת.
   - בדיקת E2E מוודאת שתוכן סריק ו-structured data קיימים בדף משחק.

## נשאר לשלב הבא

- הרחבת הטקסט הייחודי הקיים למדריכים ארוכים יותר עבור משחקים אסטרטגיים במיוחד.
- מחקר מילות מפתח בעברית עבור עמודי hub כמו “משחקי חשבון לכיתה א” ו-“תרגול שברים לילדים”.
- בדיקה בפרודקשן מול Search Console, Rich Results Test ו-Core Web Vitals.
